웹 폼태그

입력 폼을 구성하는 태그로, 사용자로부터 어떠한 값을 입력받기 위한 태그

이러한 태그는 주로 form과 form 사이에 넣어서 사용자 입력화면을 구현함

웹 폼태그의 종류

  • form: 입력받는 양식
  • labal: 라벨
  • fieldset: div 대신 구역 구분
  • input: 한 줄 입력(ex) 검색어입력란
  • select: 선택 옵션 제공 ex. 수량 선택
  • textarea: 여러 줄 입력 ex. 자기소개
  • button: 클릭 ex. 가입, 취소 검색 버튼
  • datalist: select처럼 여러 옵션 값을 취하는 목록
  • keygen: 보안키를 만들 때 사용 ex
  • output: 수학적 게산결과 출력하는 폼

각 폼태그마다 각기 다른 속성과 값을 가짐
